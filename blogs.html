Question-1: What are the different ways to select an element in the DOM?

getElementById, getElementByClassName, querySelector, querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML work with HTML and tags.
innerText work with only visible text.
textContent work with all text, ignoring CSS styles and hidden content.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is added to a parent element to handle events from its child elements through event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling means the process where an event starts from the target element and moves up to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Create = document.createElement.
Add = appendChild or append.
Remove = remove or removeChild.
